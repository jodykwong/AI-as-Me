# UX Design Specification: AI-as-Me v3.4.3

**版本**: v3.4.3  
**主题**: 统一 Dashboard + Vibe-Kanban 重构  
**日期**: 2026-01-17

---

## 1. 设计原则

1. **简洁直观** - 一眼看懂，无需学习
2. **一致性** - 统一的视觉语言和交互模式
3. **高效** - 最少点击完成任务
4. **响应式** - 适配桌面和移动端

---

## 2. 信息架构

```
首页 (/)
├── 系统概览
├── 快速操作
└── 功能导航

    ├── 灵感池 (/inspirations.html)

    ├── Soul 状态 (/soul.html)
    ├── 统计 (/stats.html)
    └── 日志 (/logs.html)
```

## 5. 交互流程

### 5.1 创建任务流程
```
1. 点击 [+ 创建任务]
2. 输入任务描述
3. 选择优先级 (默认 P2)
4. 点击 [创建]
5. 任务出现在 Inbox 列
```

### 5.2 澄清任务流程
```
1. 在 Inbox 列点击任务的 [澄清]
2. 弹出澄清表单
3. 填写目标 (必填)
4. 填写验收标准 (至少1条)
5. 可选填写工具、时间估算
6. 点击 [确认并移至 Todo]
7. 任务移动到 Todo 列
```

### 5.3 执行任务流程
```
1. 在 Todo 列点击 [开始]
2. 任务移动到 Doing 列
3. 执行完成后点击 [完成]
4. 任务移动到 Done 列
```

## 7. 响应式设计

### 桌面端 (>1024px)
- 四列并排显示

### 平板端 (768-1024px)
- 两列显示，可横向滚动

### 移动端 (<768px)
- 单列显示，Tab 切换状态
