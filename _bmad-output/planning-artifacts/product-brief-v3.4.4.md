# Product Brief v3.4.4 - Agent 模型自动化配置

## 产品概述

**版本**: v3.4.4  
**日期**: 2026-01-17  
**类型**: 功能增强

## 问题陈述

当前 AI-as-Me 集成的 OpenCode 和 Claude Code 使用硬编码的模型名称，导致：
1. 模型名称过期失效
2. 无法自动使用最新可用模型
3. 缺乏智能模型选择机制
4. 配置维护成本高

## 解决方案

实现 Agent 模型自动化配置系统：
- OpenCode: 自动检测并使用可用的免费模型
- Claude Code: 根据任务复杂度自动选择 Haiku/Sonnet/Opus

## 目标用户

- AI-as-Me 系统管理员
- 开发者
- 自动化任务执行场景

## 核心价值

1. **自动化**: 无需手动更新模型名称
2. **智能化**: 根据任务自动选择最优模型
3. **成本优化**: 优先使用免费模型
4. **可靠性**: 动态查询确保配置正确

## 成功指标

- [ ] OpenCode 自动检测成功率 100%
- [ ] Claude 模型选择准确率 > 90%
- [ ] 配置错误率降低至 0
- [ ] 模型调用成功率 > 95%

## 范围

### 包含
- OpenCode 模型动态查询
- Claude Code 模型智能选择
- 统一 Agent 管理器
- 配置文档和指南

### 不包含
- 成本追踪系统
- 多 provider fallback 机制
- Web UI 配置界面
