# Technical Research

**Date**: 2026-01-22

## Technology Stack

### Backend: FastAPI
- Modern async support
- Auto API docs
- High performance

### Agents: OpenCode + Claude Code
- OpenCode: Free, fast
- Claude Code: High quality, optional

### Storage: JSON Files + PostgreSQL
- JSON: Simple, no setup
- PostgreSQL: Optional for scale

### Frontend: Vanilla JS
- No build step
- No dependencies
- Fast loading

### Testing: pytest + Ralph
- pytest: Industry standard
- Ralph: TDD automation

## Key Decisions
- Local-first architecture
- Modular design
- Security by default
- Performance first
